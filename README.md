# [Block Breaker game](https://t73liu.github.io/block-breaker/)

## Browser Requirements

- Chrome 58+
- Firefox 53+

## Setup

1. Clone the repository.
2. Create empty 2D Unity project.
3. Copy the "Assets" and "ProjectSettings" folder to empty Unity project.
