# [The Wall](https://t73liu.github.io/the-wall/)

## Controls

- Mouse to move the paddle
- Space to launch the ball

## Setup

1. Clone the repository.
2. Open index.html using Firefox

## Tested Browsers

- Chrome 58+
- Firefox 53+
